# AI Resume Keyword Analyzer

AI Resume Keyword Analyzer is a frontend web application that compares a resume with a job description and generates an AI-style analysis.

It helps users identify keyword match score, matched skills, missing keywords, and resume improvement suggestions.

## Features

- Paste resume text
- Paste job description
- Analyze keyword match
- Generate resume match score
- Display matched keywords
- Display missing keywords
- Generate AI-style improvement suggestions
- Generate AI prompt for resume rewriting
- Save recent analysis history using local storage
- Responsive interface
- GitHub Pages ready

## Tech Stack

- HTML
- CSS
- JavaScript
- Local Storage
- GitHub Pages

## How It Works

The system extracts relevant technical keywords from both the resume and job description. It compares the keywords, calculates a match score, identifies missing skills, and provides improvement suggestions.
